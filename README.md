# Increment and Decrement Operators Practice

A beginner JavaScript practice project demonstrating the use of increment (`++`) and decrement (`--`) operators.

## Topics Covered

- Pre-increment (`++x`)
- Post-increment (`x++`)
- Pre-decrement (`--x`)
- Post-decrement (`x--`)
- Variables
- Console Output

## Purpose

This project was created to practice and understand how increment and decrement operators work in JavaScript.
